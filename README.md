# Express Server based on TypeScript & MySQL

## Tech Stack
+ TypeScript, JavaScript
+ NodeJs, Express, TypeORM, TypeDI
+ MySQL

## How To Use
1. Run `npm i` or `npm install` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm start` command
